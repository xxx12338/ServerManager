# ServerManager
Minecraft服务器管理器，使用命令提示符脚本编写。
<hr>
目前 Prismaillya Server Manager 所支持的功能：
  1.全自动化的下载所有必须文件，包括 Java，支持库等。
  2.全自动化的核心文件下载，自动下载核心支持库，自动解压文件。
  3.自建插件资源中心，一键下载你想要的插件。
  4.方便快捷的服务器管理，虽然是 cmd，但是我也能做到多窗口！
  5.服务器状态检测，当服务器关闭时自动处理。
  6.支持崩溃自动重启，告别手动重启烦恼
  7.超过21种服务端全自动下载安装。
  8.内置 BGM （ 滑稽
